# the_snake
README: Игра «Ухожор»


Описание
«Ухожор» — аркадная игра в стиле «Змейки», где вы управляете Ван Гогом. Ваша задача — собирать отрезанные в порыве вдохновения уши, избегая столкновений с собственными картинами.


Цель игры
Набрать как можно больше очков, собирая уши и создавая шедевры. 
Игра заканчивается при столкновении Ван Гога с собственной выставкой картин.


Управление
Стрелка вверх — движение вверх
Стрелка вниз — движение вниз
Стрелка влево — движение влево
Стрелка вправо — движение вправо


Системные требования
Python 3.7+
Библиотека pygame (установите через pip install pygame)
Операционная система: Windows/macOS/Linux


Установка и запуск
Скачайте репозиторий или скопируйте код.
Убедитесь, что в папке проекта есть директория images со следующими файлами:
snake_head.png — изображение головы Ван Гога
snake_body.png — изображение картин ("тела змейки")
apple.png — изображение уха (в оригинале - яблока)
background.jpg — фоновое изображение


Запустите игру командой:
bash
python ухожор.py


Особенности
Кастомизация: замените изображения в папке images, чтобы изменить внешний вид игры.
Адаптивность: игра автоматически подстраивается под размер экрана (640×480 по умолчанию).
Обработка ошибок: при отсутствии изображений игра использует цветовые заливки.
